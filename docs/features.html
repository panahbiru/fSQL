fSQL Features

This tutorial will provide a brief list of supported fSQL queries and functions. fSQL was originally designed based on mySQL so most of its syntax, features, and functions were designed after mySQL's. In the future, fSQL will have a complete set of docs detailing in the supported features but for now, this will be a good starting point.

I. Supported Queries

  • ALTER TABLE
  • BACKUP TABLE
  • BEGIN
  • COMMIT
  • CREATE TABLE
  • DELETE
  • DROP DATABASE
  • DROP TABLE
  • INSERT
  • LOCK TABLES
  • RENAME TABLE
  • REPLACE
  • RESTORE TABLE
  • ROLLBACK
  • SELECT
  • SHOW DATABASES
  • SHOW TABLES
  • START TRANSACTION
  • TRUNCATE
  • UNLOCK TABLES
  • UPDATE
  • USE
